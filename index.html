COMPUTER VISION & COMPUTER GRAPHICS

Better Visuals for Minecraft

Custom Minecraft Shader Program, May 10th, 2022

Implemented an efficient shader that gives decent shadows to the game. Minecraft is a classic game, but the default visuals are pretty basic and unrealistic. The goal of this project is to implement a shader program that provides higher graphics fidelity, in the hope of enhancing minecraft player’s gaming experience.

Cloth Simulation

Visual simulation of Cloth, April 29th, 2022

Implemented visual simulation of cloth using the classic model of point masses and strings, along with a basic shader program that renders a realistic fabric outlook. The simulation I present also involves cloth collisions with other objects (spheres and plane), and collisions with itself.
